Customer Analytics Platform for Banks & Credit Unions

Get key Insights on your customers by using Analytics Segmentation and Model

Tools and Technologies Used

  • Microsoft Fabric
  • Jupyter Notebooks
  • Power BI
  • Fabric Data Lake

Workflow

1. Data Loading and Preparation

The Fabric workspace stores data in a secure, scalable, and centralized environment. It integrates with the Data Lake for seamless access and management.

2. Data Transformation and Derived Variable Creation

Utilize Jupyter Notebooks to transform and structure data at the customer level. Apply data cleaning, aggregation, and transformation techniques for improved analysis.

3. RFM Scoring and K-Means Clustering Implementation

Segment the dataset into meaningful groups by applying RFM (Recency, Frequency, Monetary) scoring. Implement k-means clustering to identify distinct customer segments based on behavior.

4. Credit Rating Score Generation

Generate credit rating scores for customers by applying predefined conditions and scoring models. Incorporate relevant financial data and behavior patterns to assess creditworthiness.

5. Output Storage in Microsoft Fabric Data Lake

Store processed data in a structured format to ensure seamless integration with Power BI. Maintain data integrity and optimize schema design for efficient reporting.

6. Power BI Dashboard Development

Create interactive dashboards for data visualization and insights by connecting datasets from Fabric Data Lake to Power BI.

7. Publishing Dashboards in Microsoft Fabric Workspace

Publish dashboards to the Microsoft Fabric workspace, enabling secure access for authorized users. Ensure data visualization aligns with business goals and user requirements.
